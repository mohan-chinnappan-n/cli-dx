### Get SOQL for Idea
```
$ bash getObjectSOQL.sh Idea  
```
```sql
SELECT Id
    ,IsDeleted
    ,Title
    ,RecordTypeId
    ,CreatedDate
    ,CreatedById
    ,LastModifiedDate
    ,LastModifiedById
    ,SystemModstamp
    ,LastViewedDate
    ,LastReferencedDate
    ,CommunityId
    ,Body
    ,NumComments
    ,VoteScore
    ,VoteTotal
    ,Categories
    ,Status
    ,LastCommentDate
    ,LastCommentId
    ,ParentIdeaId
    ,IsHtml
    ,IsMerged
    ,CreatorFullPhotoUrl
    ,CreatorSmallPhotoUrl
    ,CreatorName
FROM Idea

```
